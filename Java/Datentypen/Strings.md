# Strings

## `String.format()` Parameter:
→ *Escape*-Zeichen: `%` und `\`
- `%s`: String
- `%d`: Digit
- `%b`: Boolean
- `%f`: Fließkommazahlen
- `%c`: Unicode-Zeichen
- `%e`: Wissenschaftliche Notation
- `%x`: Hexadezimalschreibweise
- Zeileneinzüge:
	- `%n` oder `\n`: Zeilenumbruch
	- `\t`: Tab
